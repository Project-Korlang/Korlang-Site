Quick Start

korlang new hello
cd hello
korlang run src/main.kor

Example Program

fun main() -> Int {
  let msg = "Hello, Korlang!"
  msg
}

Supported Commands

korlang new <name> [--lib|--ui|--cloud]
korlang build <file> [-o out]
korlang run <file>
korlang test
korlang doc
korlang --help
